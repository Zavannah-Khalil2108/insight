INSIGHTS

“Knowledge is useless without application.”

- Julian Hall

Simply knowing something is not enough; it must be put into practice to have value. Acquiring knowledge of HTML, the web's language, is a liberating experience. Creating websites that feature user-generated material is made possible by being proficient with its elements, tags, and characteristics. Building a website with friends to archive memories is a rewarding use of HTML knowledge. We learned how to comprehend HTML elements like headings, paragraphs, images, and links and how they work.

An enjoyable method to use HTML abilities is to create a website dedicated to memories. We created a digital library of priceless events while combining technical expertise. This project builds a meaningful area that honors friendship in addition to improving coding skills.
